Icon Placeholder File

For production use, you should create proper icons at the following sizes:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

Recommended design:
- Netflix red background (#E50914)
- White star or rating symbol
- Simple, recognizable design

You can create these icons using:
1. Design tools: Figma, Adobe Illustrator, Canva
2. Online generators: favicon.io, realfavicongenerator.net
3. AI tools: DALL-E, Midjourney with prompt "Netflix ratings extension icon"

For now, placeholder files will be created using ImageMagick or similar tools.

To create basic placeholder icons, run:
```bash
# If you have ImageMagick installed:
convert -size 16x16 xc:#E50914 -gravity center -pointsize 10 -fill white -annotate +0+0 "★" images/icon-16.png
convert -size 48x48 xc:#E50914 -gravity center -pointsize 32 -fill white -annotate +0+0 "★" images/icon-48.png
convert -size 128x128 xc:#E50914 -gravity center -pointsize 96 -fill white -annotate +0+0 "★" images/icon-128.png
```
