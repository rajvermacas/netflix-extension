KEY FINDINGS SUMMARY
====================

MOST IMPORTANT SELECTORS:
- [data-uia='video-title'] - Use this for titles
- [data-uia='video-card'] - Use this for content cards
- [role='dialog'] - Use this for modals

CRITICAL PATTERNS:
- Always use MutationObserver for detecting new content
- Always debounce mutations (100-200ms)
- Always handle React element removal
- Always use multi-fallback selectors
- Always store element references

MAJOR CHALLENGES:
1. Content script isolation (use DOM extraction)
2. React re-rendering (track and re-inject)
3. Timing issues (debounce and wait)
4. Selector changes (use multi-fallback)
5. Performance (optimize observer)
